# Examples

Each subfolder will contain a real session's outputs, anonymized where
necessary, as a worked example.

Planned:
- `mad-build-rdt/` — a copy of the RDT code-collaboration session from
  `cooperation_trial/`. Shows what MAD-build produces for a methods-code
  task.
- `mad-research-waive/` — a MAD-research audit of a working paper or
  proposal, demonstrating quote+locator discipline and the minority report.

Both will be added in the first weeks after release as we accumulate
sessions worth sharing.
